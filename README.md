# 🧠 NLP Transformers Projects Repository

This repository contains multiple Natural Language Processing (NLP) projects built using Transformer-based models from Hugging Face.  
Each project is organized in its own folder with a dedicated Jupyter Notebook and documentation.

---

## 📌 Projects Included

### 1. Named Entity Recognition (NER)
📁 `named_entity_recogniton/`

- Fine-tuned **BERT** model for Named Entity Recognition
- Detects entities such as:
  - PERSON (PER)
  - ORGANIZATION (ORG)
  - LOCATION (LOC)
- Built using Hugging Face Transformers

---

### 2. Sentiment Analysis with BERT
📁 `sentimnet_analysis_bert/`

- Text classification using **BERT**
- Predicts sentiment (positive / negative)
- Fine-tuned transformer model for NLP classification tasks

---

### 3. Text Summarization (BART)
📁 `text_summarizer_bart/`

- Abstractive text summarization using **BART**
- Converts long text into short meaningful summaries
- Built with Hugging Face pipeline
